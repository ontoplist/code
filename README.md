# OnToplist.com

Public repository for [OnToplist.com](https://www.ontoplist.com/) — an established business and blog directory. This repo is where I share code changes, document updates, and collect feedback from users.

## What This Repo Is For

OnToplist.com is maintained by a single developer. This repository exists to:

- Share code changes and feature updates in the open
- Let users report bugs and suggest improvements
- Give the community visibility into what's shipping and when

This is not a full mirror of the production codebase. It's a place for transparency, issue tracking, and discussion.

## Reporting Bugs

Found something broken? Open an issue and include:

1. **What happened** — describe the bug in one or two sentences
2. **What you expected** — what should have happened instead
3. **Steps to reproduce** — URL, browser, device, and what you clicked
4. **Screenshots** — if it's a visual issue

The more specific the report, the faster it gets fixed.

## Suggesting Improvements

Ideas are welcome. Open an issue with the `enhancement` label and describe:

- The problem you're trying to solve
- How the current site falls short
- What a better experience would look like

Not every suggestion will ship, but every one gets read.

## Tech Stack

OnToplist.com runs on a standard LAMP setup:

- Debian Linux
- Apache + PHP-FPM
- MariaDB
- Manticore Search for full-text search
- Memcached for caching
